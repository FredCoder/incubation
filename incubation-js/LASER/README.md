LASER
=====

ルールに従って、行・列の左端・上端のレーザー光点を点けたり消したりするパズルです。
----
A puzzle in which you aim to turn laser pointers on/off according to the given rules.
----

ルール：
 1. 以下のルールに従って、行・列の左端・上端にレーザー光点を点けていきます。
        （光点ありを『O』、光点無しを『X』にて表しています。）
 2. 数字は、その上下左右の最大４マスのうち、レーザーが通るマスの数を表します。
 3. 数字のあるマスは、レーザーは通りません。（数字が焼けちゃいますからね）
 4. 隣り合うレーザー光点は、同時に点けてはいけません。（干渉すると怖いからね）

Rules:
 1. each row and column either has a laser pointer ON (indicated by O) or OFF (indicated by X).
 2. a number in a grid cell denotes the number of laser beams passing adjacent to the cell.
 3. laser beams must not pass through a grid cell with a number in it. (it'll burn up the number)
 4. adjacent laser pointers must not be both on (the energy concentration would be too high to handle)
